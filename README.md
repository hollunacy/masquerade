# masquerade
Маскарад - это онлайн платформа для покупки модной одежды и аксессуаров. Сайт предоставляет возможность пользователям покупать товары, а продавцам их продавать. Основная цель проекта - создать приятное и удобное онлайн-пространство для любителей моды и ценителей прекрасного.

В нашей платформе вы можете:
- Создать аккаунт;
- Зарегистрироваться;
- Найти нужный товар через поисковую строку;
- Просмотреть карточку товара;
- Оставлять и смотреть отзывы о товарах;
- Оценивать товар и просматривать рейтинги товаров;
- Добавлять товары в избранное;
- Выбирать необходимое количество товара;
- Оформить заказ;
- Добавить товар в корзину;
- Выбрать определенные товары для взаимодействия;
- Воспользоваться различными фильтрами.
