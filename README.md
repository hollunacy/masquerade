# masquerade
Маскарад - это онлайн платформа для покупки модной одежды и аксессуаров. Сайт предоставляет возможность пользователям просматривать различные товары, добавлять товары в избранное, делать заказы. Основная цель проекта - создать приятное и удобное онлайн-пространство для любителей моды и ценителей прекрасного.

В нашей платформе вы можете:
- Создать аккаунт;
- Зарегистрироваться;
- Найти нужный товар через поисковую строку;
- Просмотреть карточку товара;
- Оставлять и смотреть отзывы о товарах;
- Оценивать товар и просматривать рейтинги товаров;
