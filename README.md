# masquerade
Маскарад - это онлайн-платформа для покупки модной одежды и аксессуаров. Сайт предоставляет возможность пользователям покупать товары, а продавцам их продавать. Основная цель проекта - создать приятное и удобное онлайн-пространство для любителей моды и ценителей прекрасного.

# В нашей платформе вы можете:
- зарегистрироваться;
- войти в существующий аккаунт;
- найти товар через поисковую строку;
- воспользоваться различными фильтрами при поиске товара;
- просмотреть карточку товара;
- перейти на страницу с товарами продавца;
- оставлять и смотреть отзывы о товарах;
- оценивать товар и просматривать рейтинги товаров;
- добавлять товары в избранное;
- добавить товар в корзину;
- выбирать необходимое количество товара для заказа;
- выбрать определенные товары для взаимодействия;
- оформить заказ;
- посмотреть историю заказов;
- поспользоваться FAQ.

Также присутствует интерфейс продавца для наиболее удобного использования онлайн-платформы. 

# Продавец может:
- зарегистрироваться;
- войти в существующий аккаунт;
- создать карточку товара;
- редактировать карточку товара;
- выставить товар на продажу;
- посмотреть историю продаж;
- воспользоваться FAQ.

# FAQ:
**Могу и я оформить заказ в интернет-магазине с доставкой по регионам Россия?**
Да.
**Нужно ли обязательно регистрироваться на сайте для оформления покупки?**
Да, вам требуется предварительно зарегистрироваться или войти в систему, чтобы совершать покупки в нашем магазине. 
**Как работает процесс оформления заказа?**
Вы находите и добавляете весь необходимый товар в корзину, нажимаете на кнопку "оформить заказ". Размещенный на нашем сайте заказ с указанной контактной информацией поступает нам сразу же. 
**Как заказать товар?**
Вы находите на сайте необходимый Вам товар, помещаете его в корзину , достаточно кликнуть по кнопке. При выборе товара для покупки в "Корзине" отобразится число товаров, помещенных в нее и сумма стоимости добавленных товаров.
Затем заходите в раздел "Корзина", при необходимости скорректируйте нужное Вам количество товара. После этого нажимаете кнопку "Оформить".
